## Description

Sonarr Adder is a Safari extension that lets you add TV shows to your Sonarr library with one click — directly from TV websites you already browse.

When you visit a TV show page on any supported site, a button appears near the title. Click it to look up the series and add it to Sonarr instantly. No need to switch apps or copy-paste titles.

Supported sites:
- IMDB
- Rotten Tomatoes
- TMDb (The Movie Database)
- TheTVDB

Features:
- One-click add to Sonarr from any supported TV show page
- Automatic series metadata detection (title, year, IMDB ID)
- Choose quality profile and root folder
- See at a glance if a show is already in your library
- Configurable Sonarr server URL and API key
- Clean, native-feeling UI that blends into each website
- Works entirely locally — no external servers, no tracking

Requires a running Sonarr instance accessible from your Mac.
