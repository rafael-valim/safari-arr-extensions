## Description

Radarr Adder is a Safari extension that lets you add movies to your Radarr library with one click — directly from movie websites you already browse.

When you visit a movie page on any supported site, a button appears near the title. Click it to look up the movie and add it to Radarr instantly. No need to switch apps or copy-paste titles.

Supported sites:
- IMDB
- Rotten Tomatoes
- TMDb (The Movie Database)
- Letterboxd
- TheTVDB

Features:
- One-click add to Radarr from any supported movie page
- Automatic movie metadata detection (title, year, IMDB ID)
- Choose quality profile and root folder
- See at a glance if a movie is already in your library
- Configurable Radarr server URL and API key
- Clean, native-feeling UI that blends into each website
- Works entirely locally — no external servers, no tracking

Requires a running Radarr instance accessible from your Mac.
