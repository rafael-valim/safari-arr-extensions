Common Questions

How do I set up the extension?

  1. Install the extension from the Mac App Store
  2. Enable it in Safari → Settings → Extensions
  3. Click the extension icon in Safari's toolbar
  4. Enter your Radarr/Sonarr server URL and API key
  5. Visit any supported website and the "Add" button will appear

Which websites are supported?

Radarr Adder (movies): IMDB, Rotten Tomatoes, TMDb, Letterboxd, TheTVDB

Sonarr Adder (TV shows): IMDB, Rotten Tomatoes, TMDb, TheTVDB

Where do I find my API key?

In your Radarr or Sonarr web interface, go to Settings → General → API Key.

The button doesn't appear on a website

  • Make sure the extension is enabled in Safari → Settings → Extensions
  • Grant the extension permission to access the website when prompted
  • Check that your Radarr/Sonarr server URL and API key are configured correctly in the extension popup
